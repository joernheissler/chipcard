.. _glossary:

Glossary
========
.. glossary::

   APDU
       Application Protocol Data Unit

   ATR
       Answer To Reset, a byte sequence sent by a ChipCard on reset
  
   FCI
       File Control Information

   ICC
       Integrated Circuit Card, a.k.a. ChipCard or SmartCard

   IFD
       Interface Device, a.k.a. ChipCard Reader

   PTS
       Protocol Type Selection

   T=0
       Character-level transmission protocol

   T=1
       Block-level transmission protocol
